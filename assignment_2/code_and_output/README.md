# 中国人口增长模型分析实验

本项目使用1949-2023年中国人口数据，建立并对比三种经典人口增长模型，通过多种参数估计方法进行拟合与预测分析。

## 数据说明

- **数据来源**：`1949-2023人口数据-实验1.xlsx`
- **数据范围**：1949年 - 2023年
- **数据内容**：中国历年人口数据（单位：万人）

## 模型列表

| 模型              | 数学公式                               | 参数说明                      |
| --------------- | ---------------------------------- | ------------------------- |
| 指数增长            | N(t) = N₀·e^(r·t)                  | N₀: 初始人口, r: 增长率          |
| 改进指数            | N(t) = N₀·e^(r·t)·(1+b·t)          | N₀: 初始人口, r: 增长率, b: 修正因子 |
| Logistic        | N(t) = K / (1 + (K/N₀-1)·e^(-r·t)) | N₀: 初始人口, r: 增长率, K: 最大容量 |
| Gompertz        | N(t) = K·e^(-b·e^(-r·t))           | K: 最大容量, b: 位移参数, r: 增长率  |
| Von Bertalanffy | N(t) = K·(1-e^(-r·(t-t₀)))³        | K: 最大容量, r: 生长率, t₀: 理论起点 |

## 参数估计方法

| 方法                  | 说明                  |
| ------------------- | ------------------- |
| 信赖域反射法(TRF)         | 通过信赖域方法求解有约束的最小二乘问题 |
| Levenberg-Marquardt | 结合梯度下降和高斯-牛顿法的优化算法  |
| Dogbox信赖域法          | 基于dogleg策略的信赖域方法    |
| BFGS拟牛顿法            | 用二阶近似加速收敛的优化算法      |

## 实验设计

### 模型检验

采用留出数据法，将数据划分为：

- **训练集**：1949 - 2010年
- **测试集**：2011 - 2023年

### 评价指标

- R²（决定系数）
- RMSE（均方根误差）
- MAE（平均绝对误差）
- MAPE（平均绝对百分比误差）

### 预测分析

对未来人口进行预测（2024-2050年），评估预测合理性。
