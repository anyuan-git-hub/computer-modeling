# 实验结论与结果分析

## 1. 实验设置说明

- 数据年份范围：1949—2023年
- 训练集：1949—2015年
- 测试集：2016—2023年
- 绘图字体自动检测结果：SimHei

## 2. 模型检验结果分析

测试集上综合表现最好的模型为 **Logistic增长模型**，对应的参数估计方法为 **Dogbox信赖域法**，其测试集 RMSE 为 **0.198977**，MAPE 为 **1.3368%**。

- Logistic增长模型：最佳方法为 Dogbox信赖域法，测试集 R2=-1.557198，RMSE=0.198977，MAPE=1.3368%
- 指数增长模型：最佳方法为 L-BFGS-B拟牛顿法，测试集 R2=-208.183167，RMSE=1.799636，MAPE=12.4795%

从测试集误差指标来看，三种模型在中国人口数据上的泛化能力存在明显差异。一般而言，若某模型在训练集上拟合较好，但在测试集上误差明显增大，则说明该模型的外推稳定性较弱；反之，则说明该模型在刻画人口增长趋势方面更稳健。

## 3. 未来人口预测结果分析

- 指数增长模型：2024年约为 15.75 亿人，2025年约为 15.93 亿人，2030年约为 16.83 亿人，2040年约为 18.79 亿人，2050年约为 20.99 亿人
- Logistic增长模型：2024年约为 14.34 亿人，2025年约为 14.39 亿人，2030年约为 14.62 亿人，2040年约为 14.97 亿人，2050年约为 15.20 亿人

在未来预测部分，代码统一给出 2024、2025、2030、2040、2050 年的人口预测值，并额外给出分阶段的**年均增长率**，避免把不同时间跨度下的累计增幅直接当作同一种增长率进行比较。

## 4. 综合结论

1. 三种模型均可用于中国人口增长趋势的刻画，但其适用范围和预测稳定性不同。
2. 指数增长模型结构最简单，但长期预测时往往更容易出现偏离现实的持续增长。
3. 改进指数增长模型在一定程度上缓解了简单指数模型的不足，但其修正项形式仍属于经验性设定。
4. Logistic 增长模型考虑了人口增长受资源与环境约束的影响，通常更适合长期趋势分析。
5. 最终应结合测试集检验指标与未来预测合理性，选取更适合作为报告主结论的模型。
